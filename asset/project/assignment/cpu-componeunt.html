10 Advantage of CPU Controls all processing and coordinates operations across the system.

12 Boolean Logic Logic circuits that handle decision-making in the CPU.
13 Graphics Concept Concepts and methods for rendering visual data through hardware.
14 Software Programs that control hardware and execute tasks on the CPU.
